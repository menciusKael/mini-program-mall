
## 项目介绍
> 该项目主要基于原生小程序开发，主要为制作小程序的项目在合成小程序部分提供模版，项目采用的微信web开发者工具v1.02
## 下载及启动
- git clone
- 开发者工具新建，无Appid可注册或者忽略体验
